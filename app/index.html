BubbleBone is a visualization of bubblesort with backbone.js

This is a simple application to demonstrate usage of backbone.js for single page applications. The concept is as follows: The script asks for a list of random integers, stored in the server under the name provided by the user. Then, the list is sorted with bubble sort and the sorted list is stored in the server. Backbone.js uses its internal event dispather to sync the list with the server by calling the REST api and animate the algorighm by updating the graphical representation of the list whenever something changes. To demo, just enter a name in the area below and click Run. It is recommended to keep the browser window side to side with the server window to watch how the list is synced.
